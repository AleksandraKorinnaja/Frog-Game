# Frogpile VS Code Project

This project is packaged from your `frogpile.html` file so you can open and edit it in Visual Studio Code.

## How to open

1. Download and unzip `frogpile-vscode-project.zip`.
2. Open Visual Studio Code.
3. Go to **File → Open Folder...** and choose the unzipped `frogpile-vscode-project` folder.
4. Open `index.html`.

## How to preview

### Option 1: Live Server extension
1. Install the **Live Server** extension in VS Code.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

### Option 2: Browser only
Double-click `index.html` or drag it into Chrome/Edge/Firefox.

## Notes

- The game is currently kept as a single HTML file with inline CSS and JavaScript.
- The `assets/` folder is included for any PNGs, icons, sounds, or exported sprites you want to add later.
- If you want, this can later be refactored into separate `style.css`, `script.js`, and asset manifest files.
